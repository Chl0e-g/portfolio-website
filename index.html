Projects

01

Backend: News App API

This project is an API which provides the backend service for a Reddit-style news site. The API interacts with a PostgreSQL database and enables CRUD functionality on articles, topics, users, and comments, as well as full error handling. I built the server with the Express package, and with full test-driven development using Jest and Supertest. The API is hosted on Heroku, and I'll be building the frontend of the application using React in the next few weeks.

02

Frontend: React Budget Planner App

This is a mini React budgeting web app I built as my first project while learning about React components and state. I designed and styled the app myself. The app makes use of form fields enabling users to add their income and as many outgoings as they wish, and this allowed me to practice using controlled components. It dynamically calculates the budget remaining based on user inputs.

03

Portfolio Website

I designed and developed this website to use as my portfolio. I made use of my UX expertise when designing the website on Figma. The site is fully responsive with a collapsible menu for mobile and tablet. I achieved this with a combination of viewport breakpoints and relative size units. It is also optimized for performance, calling in smaller graphics for smaller device sizes. While building this project I learnt Sass (SCSS) to ensure the CSS is well-organised, manageable, and easily updateable. The website is dynamic, built using a one page design with the navigation bar automatically highlighting the current section. Finally, it follows web accessibility standards.
